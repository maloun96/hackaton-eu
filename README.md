## Installation

```bash
$ npm install
```

After that, copy `.env.example` to `.env` and update it accordingly.

## Running migrations
```bash
npm run migration:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Redis 

```bash
$ brew install redis

$ brew services start redis
```
